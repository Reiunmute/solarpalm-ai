# Static website

This directory contains a polished static landing page for SolarPalm AI.

## Preview locally

```bash
cd site
python3 -m http.server 4173
```

Then open `http://127.0.0.1:4173/`.

## Scope

- Pure HTML, CSS, and JavaScript.
- No package manager or build step.
- Keeps the Python package unchanged.
- The browser calculator gives a simple front-end estimate for local preview.
